# spatula merge-matched-tags

## Summary

`spatula merge-matched-tags` merges the per-batch matched-tag output produced by the [match-tag](match_tag.md) command into a single tag DGE (digital gene expression) file. It takes a list of `match-tag` output files and the tag dictionary, and writes the combined tag-by-barcode counts to the output directory.

## Required Options

* `--list STR` : List of output files from `match-tag`.
* `--tag STR` : Dictionary file containing tag sequences.
* `--out STR` : Output directory to store the DGE file.

## Expected Output

* `[out]/` : Output directory containing the merged tag DGE.
