# Tools in spatula

`spatula` is a collection of C++ tools that are designed to help analyze 
sub-micron resolution
spatial transcriptomics data such as Seq-Scope. 
The categories of software tools provided includes:

* Seq-Scope Tools
* [FICTURE](https://github.com/seqscope/ficture) Tools
* FASTQ Tools
* Image Tools
* Other Tools

!!! note
    This documentation contains only a subset of primary tools that are used by other
    tools or pipelines including [FICTURE](https://github.com/seqscope/ficture),
    [FICTURE2](https://github.com/Yichen-Si/punkst), 
    [Cartloader](https://github.com/seqscope/cartloader), and
    [NovaScope](https://github.com/seqscope/novascope),
    Other tools not documented here are considered non-primary tools that have limited support, and they are NOT supported officially.
